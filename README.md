# Turtle

This is "turtle" mod for Luanti; it provides programmable robots called
"turtles", which can move and make other automated actions such as digging
or placing nodes.

## Dependencies
* Luanti version 5.10+
* [digilines](https://github.com/minetest-mods/digilines)
* [pipeworks](https://github.com/mt-mods/pipeworks)
